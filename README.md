# 🖼️ Text-to-Image Generation using Stable Diffusion

## 📌 Overview

This project demonstrates how to generate high-quality images from text prompts using Stable Diffusion. The implementation is done in Google Colab with GPU acceleration.

## 🚀 Features

* Text-to-image generation using natural language prompts
* High-resolution image outputs
* GPU support for faster processing
* Easy-to-use pipeline

## 🛠️ Tech Stack

* Python
* PyTorch
* Hugging Face Diffusers
* Google Colab

## ⚙️ How It Works

1. Install required libraries
2. Load pre-trained Stable Diffusion model
3. Input a text prompt
4. Generate and save the image
